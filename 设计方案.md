# 模拟计算器

## 主要功能

- RC 滤波器截止频率计算
  - 支持多级 RC 串联，可动态增加/删除级数
  - 每级独立输入 R、C，实时计算 fc = 1/(2πRC)
  - 前端本地计算，无需调用后端

- 运放 Vrms 噪声计算
  - 输入 GBW、增益、Vn_density
  - 允许选择输出滤波器：1 阶 RC、2 阶 RC、3 阶 RC
  - 输入 RC 滤波器带宽
  - 后端 Rust 计算：考虑闭环带宽 min(GBW/Gain, ENB × RC_BW)

- 电阻分压计算器
  - 支持 E6/E12/E24/E96 系列筛选
  - 允许选择是否串并联组合
  - 输入 Vi 和目标 Vo，计算多个优化方案
  - 输出方案包含 R₁、R₂、实际 Vo、误差百分比
  - 后端 Rust 搜索：遍历 E 系列标准值（1Ω~10MΩ），取误差 < 5% 的前 10 个

## 通用特性

- 输入框支持单位后缀：p、n、u/µ、m、k/K、M、G
  - 例如 "1k"=1000、"10n"=1e-8、"2.2M"=2200000
  - 同时兼容纯数字和科学记数法（如 "1e-6"）
- 输出自动按 SI 前缀格式化（如 1.59 kHz、10.00 µV、4.70 kΩ）
- 所有输入框失去焦点时自动计算，无计算按钮
- 组件挂载时自动执行首次计算

## 设计选型

- Rust 2024 edition
- Tauri 2.0
- React 19
- TypeScript
- Tailwind CSS v4

## 设计要点

- 添加依赖时需要考虑使用最新版本，通过命令行添加而不是直接修改文件
- 根据 cargo/pnpm 等渠道确定最新版本
- 纯计算逻辑（RC 截止频率）在前端直接执行；涉及搜索和复杂运算（噪声计算、分压搜索）放在 Rust 后端
