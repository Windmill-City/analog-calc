# 模拟计算器

## 主要功能

- 低通/高通滤波器截至频率计算
  - 输入 RC 值
- 运放Vrms噪声计算
  - 输入GBW, 增益， Vn_density
  - 允许选择输出滤波器 1阶RC, 2阶RC，3阶RC，输入RC滤波器带宽
- 电阻分压计算器
  - 允许 E6/E12/E24/E96 筛选
  - 允许选择是否串并联
  - 输入Vi/Vo计算多个方案
  - 输出方案需要计算Vi固定时Vo/Vo固定时Vi

## 设计选型

- Rust 2024
- Tauri 2.0
- React
- Tailwind CSS

## 设计要点

- 添加依赖时需要考虑使用最新版本, 通过命令行添加而不是直接修改文件
- 根据 cargo/pnpm 等渠道确定最新版本
