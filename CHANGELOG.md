# Changelog

## [1.0.0] - 2026-05-13

### Added

- 电阻分压：增加最大误差滑动条，支持串联/并联扩展，可配置权重评分
- 电阻取值：从 E6/E12/E24/E96 系列中搜索最接近的目标电阻值
- 等效噪声带宽（ENBW）及总带宽显示
- 持久化存储（zustand）
- CI/CD：GitHub Actions 自动构建与发布（支持 Linux/macOS/Windows）

- 全面优化用户界面
- 优化串并联逻辑与搜索性能
- 输入框回车直接提交
